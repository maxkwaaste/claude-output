How to Read This

Field Reference

Focus keyword — the Yoast focus keyphrase field. The primary query this page should rank for.

Title tag — what appears in the browser tab and Google SERP title. Limit: ~60 characters.

Meta description — the snippet below the title in Google. Limit: ~155 characters.

GSC data — impressions = how often Google showed proxuma.io for this query over 16 months. Position = average SERP position. These are proxies for total search volume.

In the tables below: green = keep as-is   red = change needed
