Bootstrap Collision Warning

Bootstrap class names .card, .card-header, .badge, .btn, .nav-*, .form-*, .modal-*, .alert-*, .table-* all leak Bootstrap defaults into custom page content.

Always prefix custom classes (e.g., .dxf-card, .wl-card) to avoid collisions. When overriding Bootstrap classes, explicitly reset: padding:0; background:transparent; border:0; border-radius:0; color:inherit.

Visual Identity

Dark-themed, bold, high-contrast MSP consultancy. Dark backgrounds (#030303 to #1F1F1F) with white text. A single red-to-orange gradient (#ED4C35 to #FF6927) serves as the only accent. PP Monument uppercase headings feel blocky and authoritative. DM Sans body text is clean and modern. Minimal border-radius (2-4px) keeps everything sharp and professional. Trapezoid clip-paths are the signature visual element. Animations are subtle entrance effects, never flashy. The overall mood: serious, expert, no-nonsense.
